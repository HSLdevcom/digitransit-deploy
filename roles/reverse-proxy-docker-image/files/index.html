Release timeline

Alpha version

1st Mar 2015 - 31st Aug 2015

  1. Test environment Create a test environment.
  2. Routing v.1 Simple routing algorithm.
  3. UI Concept design Create first concept design of
    mobile UI.
  4. Geocoding service v.1 Create service for geocoding and
    reverse geocoding.
  5. Map service v.1 Create nationwide basemap and
    styles for it.
  6. Route data v.1 Create services for loading
    nationwide route data.
  7. Real-time service v.1 Create initial support for
    realtime data.

Beta version

1st Sep 2015 - 28th Feb 2016

  1. Improved UI Improved UI.
  2. Route data v.2 Improved route data service.
  3. Real-time service v.2 - LIJ Improved real-time data service
    based on LIJ.
  4. Routing v.2 Improved routing algorithm.
  5. Park and ride needs Park and ride needs.
  6. Production environment Production environment.

Release version

1st Mar 2016 - 31st Aug 2016

  1. Improved UI Improved UI.
  2. Route data v.3 Improved route data service.
  3. Real-time service v.3 Improved real-time data service.
  4. Routing v.3 Improved routing algorithm.
  5. API management API management.
  6. Interpreting old API calls Interpreting old API calls.
  7. Final UI Finalized Reittiopas
    User interface.
